Dateien herunterladen

2023-02-28

Die benötigte Dateien für diesen Kurs stehen zur Verfügung unter folgenden Link.

Um diese Dateien direkt in deinem Arbeitsverzeichnis zu speichern, führe folgende Befehle in deiner Sitzung:

download.file(url = "https://kamapu.github.io/GrundkursR/documents/KursDateien.zip",
    destfile = "KursDateien.zip", method = "curl")
unzip("KursDateien.zip", overwrite = TRUE)
unlink("KursDateien.zip")
